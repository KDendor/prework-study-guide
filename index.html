HTML

  • Stands for Hyper Text Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • One of three cornerstone languages in web dev
  • Used to create the structure of a webpage
  • Made up of elements that work together to provide content and basic framework
  • HTML is like the sturdy frame of a house
  • HTML elements are usually composed of an opening and closing tag
  • Tags contain everything that we want to include in the element
  • Elements are a type of content on a webpage
  • Not all elements are visible to users

CSS

  • Programming language that defines how webpage content is presented to the user
      • Defines the colors, aesthetics, visual layout, and organization

  • There are three ways to style a webpage using CSS
      • Inline CSS
      Internal CSS style sheet
      External CSS style sheet

  • A selector defines the elements or attributes to which the rules, or declarations, will apply
      • Declarations contain two important components: the CSS property we want to apple and the value of the property
  • Margin property acts like the margin in a word doc
      • A margin indicates how much space we want around the outside of an element
      Margin set to zero means no extra space around the element
  • A padding indicates how much space we want around the content inside and element

  • CSS box model: Every HTML element is essentially a box which consists of margins outside the element and the element's border, padding, and content

  • asterik * applies a rule to all elements visible on the page

JavaScript

  • JS is a dynamic programming language that allows users to interact with websites
      • Control flow is the order in which a computer executes code in a script
      The browser reads code in the order that it appears in the file
  • A variable is a named container that allows us to store data in our code
      • You start by declaring a variable with a keyword
      A semicolon at the end of a line indicates where a statement ends
  • JS is case sensitive
  • String
      • A sequence of text
      enclosed in single or double quotation marks
  • Number
      • a number
  • Boolean
      • True/False value
  • Array
      • A structure that allows you to store multiple values in a single reference
  • Object
      • Can be anything
      Everything in JS is an object
